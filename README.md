# 🌿 Natours – Landing Page

### בנוי בעזרת **HTML | CSS | SASS**

פרויקט זה הוא דף נחיתה מודרני בסגנון אתר טיולים, שנבנה כחלק מלימוד SASS וארגון קוד מקצועי. האתר משלב עיצוב נקי, אנימציות, שימוש במשתנים, mixins, partials ומבנה תיקיות מודולרי.

---

## 🚀 תכונות הפרויקט

* שימוש ב־**SASS** במקום CSS רגיל
* מבנה תיקיות מודולרי:

  * `abstracts/` – משתנים, פונקציות, mixins
  * `base/` – resets, טיפוגרפיה, utilities
  * `components/` – כפתורים, כרטיסים, טפסים
  * `layout/` – header, footer, grid
  * `pages/` – עיצוב עמודים ספציפיים
* Responsive מלא עם Media Queries דרך Mixins
* אנימציות עדינות למרכיבים כמו טקסט, כפתורים וניווט
* שימוש ב־BEM לכתיבת CSS נקי וניתן לתחזוקה
* תמיכה בפונט מותאם (Open Sans / Google Fonts)

## 🛠️ התקנה והרצה

### 1️⃣ התקנת חבילות

```bash
npm install
```

### 2️⃣ הידור Sass לקובץ CSS

```bash
npm run watch:sass
```

### 3️⃣ הרצת Live Server

```bash
npm run devserver
```

או להרצה משולבת:

```bash
npm start
```

## 🎯 מה נלמד בפרויקט?

הפרויקט מדגים:

* שימוש נכון ב־**SASS** וכתיבה מודולרית
* בניית משתנים לעיצוב עקבי
* יצירת mixins למידתיות, גריד ו־Media Queries
* כתיבת CSS נקי בעזרת BEM
* בניית עמוד נחיתה מלא הכולל ניווט, Header, כפתורים, כרטיסים ו־Footer
* יצירת עיצוב מודרני עם גרדיאנטים, תמונות רקע ואנימציות
